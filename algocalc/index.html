Алгоритмический вычислитель - проект создания искусственного интеллекта общего алгоритмического вида.



ОПИСАНИЕ ТЕХНОЛОГИИ

Машинный интеллект - "супералгоритм", реализованный в виде программного комплекса, способного автоматически формировать типы данных и находить алгоритмы над этими типами для решения метаэвристических задач, т.е. задач, заданных динамически в виде отклика фитнес-функции.

Эволюционные методы [7-10] – методы стохастической оптимизации, при которой используются базовые принципы биологической эволюции – отбор, мутация, адаптация и воспроизводство. Адаптационные методы усиливают эволюционную модель, предоставляя возможность формирования в процессе эволюции у программных агентов популяции дополнительных полезных навыков (библиотек), которые могут быть непосредственно переданы в следующее поколение

Технология может использоваться в различных областях, например в машинном обучении, где она способна помочь преодолеть некоторые трудности современного направления глубокого обучения.

ПРОГРАММНАЯ РЕАЛИЗАЦИЯ

COMPOT (Common Program Optimisation Toolkit) - это Open Source реализация описанной технологии на языке Tcl/Tk – скриптовом языке быстрого прототипирования.

Плюсы: гибкость, простота, мало синтаксического сахара, нативный GUI, простая парадигма «все есть строка», которая хорошо ложится на идею алгоритмических преобразований.
Минусы: не высокая скорость исполнения, что является обратной стороной его гибкости на этапе выполнения.
