Алгоритмическое исчисление - методика и проект создания машинного интеллекта общего алгоритмического вида для использования в частных задачах поиска оптимальных решений.



ОПИСАНИЕ ТЕХНОЛОГИИ

Машинный интеллект в нашем проекте - это "супералгоритм" (реализованный в виде программного комплекса), способный автоматически формировать необходимые типы данных и, одновременно, находить алгоритмы над этими типами для решения метаэвристических задач, т.е. задач, заданных динамическим описанием функции "пригодности".

Основу программного комплекса составляют:

  • Эволюционные методы [7-8] – методы стохастической оптимизации c использованием базовых принципов биологической эволюции (отбор, мутация, адаптация и воспроизводство).
  • Адаптационные методы [9-10] – усиливают эволюционную модель, предоставляя возможность формирования в процессе эволюции у программных агентов популяции дополнительных полезных навыков (библиотек), которые могут быть непосредственно переданы в следующее поколение

    Технология может использоваться в различных областях, например в машинном обучении, где она, возможно, будет способствовать преодолению некоторых ограничений в современных алгоритмах глубокого обучения.

    ПРОГРАММНАЯ РЕАЛИЗАЦИЯ

    COMPOT (Common Program Optimisation Toolkit) - это Open Source реализация описанной технологии на языке Tcl/Tk – скриптовом языке быстрого прототипирования.

    Плюсы: гибкость, простота, мало синтаксического сахара, нативный GUI, простая парадигма «все есть строка», которая хорошо ложится на идею алгоритмических преобразований.
    Минусы: не высокая скорость исполнения, что является обратной стороной его гибкости на этапе выполнения.
